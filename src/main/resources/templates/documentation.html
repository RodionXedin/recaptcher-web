API

* You can not specify your own proxies, this functionality will be added soon.

1. How to create a task: 

POST https://api.recaptcher.com/api/v1/createTask
{
    "clientKey":"8bef160a-0aa5-4505-871f-1d53f748c4cb", // Your API KEY that you can find at Settings page
    "task":
        {
            "websiteURL":"https://www.google.com/recaptcha/api2/demo", // you need to specific a page with reCAPTCHA, not just link to website
            "websiteKey":"6Le-wvkSAAAAAPBMRTvw0Q4Muexq9bi0DJwx_mJ" // reCAPTCHA site-key
        }
}

Response example:
{
  "taskId": "YW50aWNhcHRjaGE=366275548"
}

2. How to get reCAPTCHA answer: 

POST https://api.recaptcher.com/api/v1/getTaskResult
{
    "clientKey":"8bef160a-0aa5-4505-871f-1d53f748c4cb",
    "taskId":"YW50aWNhcHRjaGE=366275545"
}

Response example:
{
  "answer": "03AHJ_VuuluIjnKdVHW1-0zAg9IJlvGtxQLSJ...",
  "ready": true
}

Or if an answer is not ready yet:
{
  "answer": null,
  "ready": false
}
